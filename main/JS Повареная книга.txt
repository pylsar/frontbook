alert выводит на экран окно с сообщением и приостанавливает выполнение скрипта, пока пользователь не нажмёт «ОК».
prompt - 	result = prompt(title, default); 
	выводит модальное окно с заголовком title, полем для ввода текста, заполненным строкой по умолчанию default и кнопками OK/CANCEL.(Всегда указывайте default).
	Вызов prompt возвращает то, что ввёл посетитель – строку или специальное значение null, если ввод отменён.
	пример: 
		var years = prompt('Сколько вам лет?', 100);
		alert('Вам ' + years + ' лет!')

confirm -	result = confirm(question);
	confirm выводит окно с вопросом question с двумя кнопками: OK и CANCEL. Результатом будет true при нажатии OK и false – при CANCEL(Esc).
	Пример:
		var isAdmin = confirm("Вы - администратор?");
		alert( isAdmin );

window.onload = function(e){} - произвести какое либо действие после загрузки виндовс
когда у нас на элементе висит уже одно событие пример: document.body.onclick = function(e){}, то мы не можем на него повесить ещё одно событие. Тогда
используем addEventListener   пример: document.body.addEventListener("click", function(){});
